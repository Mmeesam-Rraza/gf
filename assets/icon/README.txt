LAUNCHER ICON SETUP
===================

1. Place your app icon image here as: app_icon.png
2. Recommended size: 1024x1024 pixels (PNG format)
3. Run: flutter pub run flutter_launcher_icons

The package will automatically generate all required sizes:
- mipmap-hdpi (72x72)
- mipmap-mdpi (48x48)
- mipmap-xhdpi (96x96)
- mipmap-xxhdpi (144x144)
- mipmap-xxxhdpi (192x192)

For now, if you build without an icon, Flutter will use its default icon.
